# 🎮 <span style="color:#4CAF50;">Programa del Juego del Ahorcado</span>

---

## 📝 <span style="color:#2196F3;">Descripción</span>

<p style="color:#E91E63;">
Por medio de este repositorio, se presentó un programa con múltiples menús de entrada de datos,
en donde se podrán agregar productos y también jugar el clásico juego del ahorcado,
con temática de productos de una tienda comercial.
</p>

---

## ⚙️ <span style="color:#FF9800;">Funcionalidades</span>

<ul>
  <li style="color:#9C27B0;">📦 Gestión de inventario (agregar, mostrar, buscar y ordenar productos)</li>
  <li style="color:#009688;">🎯 Juego del ahorcado interactivo</li>
  <li style="color:#3F51B5;">🔁 Uso de recursividad para control de intentos</li>
</ul>
